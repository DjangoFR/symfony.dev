# symfony.dev
Symfony on Docker

First, clone this repository:

```bash
$ git clone git@github.com:DjangoFR/symfony.dev.git
```

# Tools you need on your host


```text
docker
docker-compose (>= 1.6)
  -> run ./bin/docker-compose-install.sh
wget
phpstorm
```

# Add following to /etc/hosts
127.0.0.1	symfony.dev

# Installation

$ ./bin/build.sh 
$ ./bin/start.sh 
$ ./bin/symfony.sh
